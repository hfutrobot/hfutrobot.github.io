常微分方程求解之梯形法

梯形法是求解常微分方程初值问题的重要隐式数值方法。本文将详细推导其基本公式，并严格证明截断误差、收敛性和稳定性，特别是其优越的A-稳定性质。

1. 问题背景

考虑常微分方程初值问题 (Initial Value Problem, IVP)：

\[ \begin{cases} \frac{dy}{dx} = f(x, y) \\\\ y(x_0) = y_0 \end{cases} \]

微分方程 \(y' = f(x, y)\) 可以改写为积分形式：

\[ y(x_{n+1}) = y(x_n) + \int_{x_n}^{x_{n+1}} f(x, y(x)) \, dx \]

核心想法：用梯形公式近似这个积分。

梯形积分公式：

\[ \int_a^b g(x) \, dx \approx \frac{b - a}{2} [g(a) + g(b)] \]

应用到我们的问题（\(h = x_{n+1} - x_n\)）：

\[ \int_{x_n}^{x_{n+1}} f(x, y(x)) \, dx \approx \frac{h}{2} [f(x_n, y(x_n)) + f(x_{n+1}, y(x_{n+1}))] \]

因此：

\[ y(x_{n+1}) \approx y(x_n) + \frac{h}{2} [f(x_n, y(x_n)) + f(x_{n+1}, y(x_{n+1}))] \]

用数值解 \(y_n\) 代替精确解 \(y(x_n)\)，得到梯形公式：

\[ \boxed{y_{n+1} = y_n + \frac{h}{2} [f(x_n, y_n) + f(x_{n+1}, y_{n+1})]} \]

几何意义：用梯形面积代替曲线下面积，比Euler法（矩形面积）更精确。

2.2 隐式特性

注意 \(y_{n+1}\) 同时出现在等式两边（在 \(f(x_{n+1}, y_{n+1})\) 中），这是隐式方法。

求解方法：

  1. 线性情况（如 \(y' = \lambda y\)）：直接代数求解

    \[ y_{n+1} = y_n \cdot \frac{1 + \frac{h\lambda}{2}}{1 - \frac{h\lambda}{2}} \]

  2. 非线性情况：使用不动点迭代或牛顿法

    \[ y_{n+1}^{(k+1)} = y_n + \frac{h}{2}[f(x_n, y_n) + f(x_{n+1}, y_{n+1}^{(k)})] \]

3. 截断误差分析

3.1 基于积分余项的证明

定义：假设第 \(n\) 步是精确的（\(y_n = y(x_n)\)），一步计算引入的误差：

\[ R_{n+1} = y(x_{n+1}) - y_{n+1} \]

由于梯形法是基于梯形积分公式构造的，我们可以直接利用数值积分的余项定理来推导其截断误差。

考察微分方程的积分形式：

\[ y(x_{n+1}) - y(x_n) = \int_{x_n}^{x_{n+1}} y'(x) \, dx \]

梯形法的数值近似公式为：

\[ y_{n+1} - y_n = \frac{h}{2} [f(x_n, y_n) + f(x_{n+1}, y_{n+1})] \]

在计算局部截断误差时，假设 \(y_n = y(x_n)\)，则 \(f(x_n, y_n) = y'(x_n)\)。 因此，局部截断误差 \(R_{n+1}\) 等于积分真值与梯形近似值之差：

\[ R_{n+1} = \int_{x_n}^{x_{n+1}} y'(x) \, dx - \frac{h}{2} [y'(x_n) + y'(x_{n+1})] \]

根据微积分中的梯形公式余项定理：

\[ \int_a^b g(x) \, dx - \frac{b-a}{2}[g(a) + g(b)] = -\frac{(b-a)^3}{12} g''(\xi), \quad \xi \in (a, b) \]

令 \(g(x) = y'(x)\)，区间为 \([x_n, x_{n+1}]\)，则 \(b-a = h\)。代入公式得：

\[ R_{n+1} = -\frac{h^3}{12} \frac{d^2}{dx^2}(y'(x)) \bigg|_{x=\xi} = -\frac{h^3}{12} y'''(\xi) \]

结论：

\[ \boxed{R_{n+1} = -\frac{h^3}{12} y'''(\xi) = O(h^3)} \]

这证明了梯形法的局部截断误差为 3 阶，因此它是 2 阶精度 的方法（整体误差比局部误差低一阶）。

3.2 整体截断误差的严格推导

问题的核心：为什么局部截断误差是 \(O(h^3)\)，整体误差却是 \(O(h^2)\)？

这是因为误差会在 \(N = (X - x_0)/h\) 步中累积。下面给出严格的数学证明。

3.2.1 误差递推关系

设 \(e_n = y(x_n) - y_n\) 为第 \(n\) 步的整体误差（global error）。

精确解满足：

\[ y(x_{n+1}) = y(x_n) + \frac{h}{2}[f(x_n, y(x_n)) + f(x_{n+1}, y(x_{n+1}))] + R_{n+1} \]

其中 \(R_{n+1} = -\frac{h^3}{12}y'''(\xi_n)\) 是局部截断误差。

数值解满足：

\[ y_{n+1} = y_n + \frac{h}{2}[f(x_n, y_n) + f(x_{n+1}, y_{n+1})] \]

两式相减，得到误差递推式：

\[ e_{n+1} = e_n + \frac{h}{2}\left\{ [f(x_n, y(x_n)) - f(x_n, y_n)] + [f(x_{n+1}, y(x_{n+1})) - f(x_{n+1}, y_{n+1})] \right\} + R_{n+1} \]

3.2.2 Lipschitz 条件的应用

假设：\(f(x,y)\) 关于 \(y\) 满足 Lipschitz 条件，即存在常数 \(L > 0\) 使得：

\[ |f(x, y_1) - f(x, y_2)| \leq L|y_1 - y_2| \]

应用此条件到误差递推式：

\[ |e_{n+1}| \leq |e_n| + \frac{h}{2}\left[ L|e_n| + L|e_{n+1}| \right] + |R_{n+1}| \]

整理得：

\[ |e_{n+1}| \leq |e_n| + \frac{hL}{2}|e_n| + \frac{hL}{2}|e_{n+1}| + |R_{n+1}| \]

移项（将 \(|e_{n+1}|\) 项移到左边）：

\[ |e_{n+1}|\left(1 - \frac{hL}{2}\right) \leq |e_n|\left(1 + \frac{hL}{2}\right) + |R_{n+1}| \]

假设步长充分小使得 \(h < \frac{2}{L}\)，则 \(1 - \frac{hL}{2} > 0\)，可以除以它：

\[ |e_{n+1}| \leq |e_n| \cdot \frac{1 + \frac{hL}{2}}{1 - \frac{hL}{2}} + \frac{|R_{n+1}|}{1 - \frac{hL}{2}} \]

3.2.3 放缩与化简

定义增长因子：

\[ \rho = \frac{1 + \frac{hL}{2}}{1 - \frac{hL}{2}} \]

利用泰勒展开 \(\frac{1}{1-x} = 1 + x + O(x^2)\)：

\[ \rho = \left(1 + \frac{hL}{2}\right)\left(1 + \frac{hL}{2} + O(h^2)\right) = 1 + hL + O(h^2) \]

因此，当 \(h\) 充分小时：

\[ \rho \leq 1 + hL + Ch^2 \leq e^{hL} \]

（利用了 \(1+x \leq e^x\) 的不等式）

同时，由于 \(|R_{n+1}| \leq \frac{h^3}{12}M_3\)，其中 \(M_3 = \max |y'''(x)|\)，故：

\[ \frac{|R_{n+1}|}{1 - \frac{hL}{2}} \leq \frac{h^3 M_3/12}{1 - hL/2} \leq \frac{h^3 M_3}{12(1 - hL/2)} \leq C h^3 \]

3.2.4 递推求解

从 \(e_0 = 0\) 开始，递推可得：

\[ \begin{align} |e_1| &\leq \rho \cdot 0 + Ch^3 = Ch^3 \\\\ |e_2| &\leq \rho \cdot Ch^3 + Ch^3 = Ch^3(1 + \rho) \\\\ |e_3| &\leq \rho^2 Ch^3 + \rho Ch^3 + Ch^3 = Ch^3(1 + \rho + \rho^2) \\\\ &\vdots \\\\ |e_n| &\leq Ch^3(1 + \rho + \rho^2 + \cdots + \rho^{n-1}) \end{align} \]

利用几何级数求和公式：

\[ 1 + \rho + \rho^2 + \cdots + \rho^{n-1} = \frac{\rho^n - 1}{\rho - 1} \]

由于 \(\rho \leq e^{hL}\)，且 \(nh = x_n - x_0 \leq X - x_0\)（有界），故：

\[ \rho^n \leq e^{nhL} = e^{(x_n - x_0)L} \]

这是一个与 \(h\) 无关的常数！因此：

\[ |e_n| \leq Ch^3 \cdot \frac{e^{(x_n-x_0)L} - 1}{\rho - 1} \]

现在关键是估计 \(\rho - 1\)。从前面的展开：

\[ \rho - 1 = hL + O(h^2) \geq hL/2 \quad (\text{当 } h \text{ 充分小时}) \]

代入得：

\[ |e_n| \leq Ch^3 \cdot \frac{e^{(x_n-x_0)L}}{hL/2} = \frac{2C}{L} \cdot e^{(x_n-x_0)L} \cdot h^2 \]

结论：

\[ \boxed{|e_n| = |y(x_n) - y_n| \leq K h^2} \]

其中 \(K = \frac{2M_3}{12L} \cdot e^{(X-x_0)L}\) 是与 \(h\) 无关的常数。

物理意义：

  • 局部误差 \(O(h^3)\) 表示单步引入的误差
  • 整体误差 \(O(h^2)\) 表示所有步累积后的误差
  • 从 \(x_0\) 到 \(x_n\) 总共需要 \(n = O(1/h)\) 步
  • 累积效应：\(O(h^3) \times O(1/h) = O(h^2)\)

这就是局部截断误差阶数 = 整体误差阶数 + 1 的数学原因！

4. 收敛性分析

4.1 收敛性的定义

定义 4.1 (收敛性)
如果当步长 \(h \to 0\) 时，对固定的 \(x_n = x_0 + nh\)（保持 \(x_n\) 固定），有：

\[ \lim_{h \to 0} y_n = y(x_n) \]
则称该数值方法是收敛的。

4.2 收敛性定理

定理 4.1 (梯形法的收敛性)
若函数 \(f(x, y)\) 满足 Lipschitz 条件，且精确解 \(y(x) \in C^3[a, b]\)，则梯形法是收敛的，且整体截断误差满足：

\[ |y(x_n) - y_n| \leq K h^2 \]
即梯形法具有 2 阶收敛性。

证明：

由第 3.2 节的严格推导可知，整体误差满足：

\[ |y(x_n) - y_n| \leq K h^2 \]

其中 \(K = \frac{2M_3}{12L} \cdot e^{(X-x_0)L}\) 是与 \(h\) 无关的常数。

现在证明收敛性。对于固定的 \(x_n = x_0 + nh\)，当 \(h \to 0\) 时（相应地 \(n \to \infty\) 保持 \(nh\) 固定）：

\[ \lim_{h \to 0} |y(x_n) - y_n| \leq \lim_{h \to 0} K h^2 = 0 \]

因此：

\[ \lim_{h \to 0} y_n = y(x_n) \]

根据定义 4.1，梯形法是收敛的。

此外，由于误差界为 \(O(h^2)\)，梯形法具有 2 阶收敛性。 \(\square\)

4.3 收敛阶

由误差估计式可知：

\[ \max_{0 \leq n \leq N} |y(x_n) - y_n| = O(h^2) \]

因此梯形法是2阶收敛的，收敛速度比Euler法（1阶）快得多。

实际意义：若要达到相同精度，梯形法的步长可以是Euler法的 \(\sqrt{10} \approx 3.16\) 倍，计算量大大减少！

5. 绝对稳定性分析

5.1 测试方程与增长因子

测试方程（Dahlquist test equation）：

\[ \frac{dy}{dx} = \lambda y, \quad y(0) = y_0, \quad \lambda < 0 \]

精确解为 \(y(x) = y_0 e^{\lambda x}\)，当 \(x \to \infty\) 时，\(y(x) \to 0\)。

应用梯形法：

\[ y_{n+1} = y_n + \frac{h}{2}[\lambda y_n + \lambda y_{n+1}] \]

整理可得：

\[ y_{n+1} = y_n \cdot \frac{1 + \frac{h\lambda}{2}}{1 - \frac{h\lambda}{2}} \]

定义增长因子 \(G(z)\)，其中 \(z = h\lambda\)：

\[ \boxed{G(z) = \frac{1 + \frac{z}{2}}{1 - \frac{z}{2}}} \]

5.2 A-稳定性证明

定义 5.1 (绝对稳定域)
对于数值方法，使得 \(|G(z)| \leq 1\) 的 \(z = h\lambda\) 的集合称为绝对稳定域。

定义 5.2 (A-稳定性)
如果数值方法的绝对稳定域包含整个左半复平面（即所有 \(\text{Re}(z) < 0\)），则称该方法是 A-稳定的 (A-stable)。

定理 5.1 (梯形法的A-稳定性)
梯形法是A-稳定的。

证明： 需要证明：对所有 \(z = x + iy\)，\(x < 0\)，有 \(|G(z)| \leq 1\)。

\[ |G(z)|^2 = \left|\frac{1 + \frac{z}{2}}{1 - \frac{z}{2}}\right|^2 = \frac{\left|1 + \frac{x}{2} + i\frac{y}{2}\right|^2}{\left|1 - \frac{x}{2} - i\frac{y}{2}\right|^2} = \frac{(1 + \frac{x}{2})^2 + (\frac{y}{2})^2}{(1 - \frac{x}{2})^2 + (\frac{y}{2})^2} \]

展开分子分母：

\[ \text{分子} = 1 + x + \frac{x^2}{4} + \frac{y^2}{4} \]
\[ \text{分母} = 1 - x + \frac{x^2}{4} + \frac{y^2}{4} \]

当 \(x < 0\) 时，显然 \(1+x < 1-x\)，故分子 < 分母，即 \(|G(z)|^2 < 1\)。 因此梯形法是 A-稳定的。 \(\square\)

5.3 与Euler法的比较

方法 增长因子 \(G(z)\) 稳定域 A-稳定性
显式Euler \(1 + z\) \(\|1 + z\| \leq 1\)
(圆盘区域)		 否
隐式Euler \(\frac{1}{1 - z}\) \(\left\|\frac{1}{1-z}\right\| \leq 1\)
(左半平面)		 是
梯形法 \(\frac{1 + \frac{z}{2}}{1 - \frac{z}{2}}\) \(\text{Re}(z) < 0\)
(左半平面)		 是

结论：梯形法不仅精度比隐式欧拉法高（2阶 vs 1阶），而且同样拥有完美的A-稳定性，是解决刚性问题的优秀选择。
