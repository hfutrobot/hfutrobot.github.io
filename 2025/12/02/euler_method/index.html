常微分方程求解之欧拉法

Euler法是求解常微分方程初值问题最基本的数值方法。本文将详细推导其基本公式,并严格证明截断误差、收敛性和稳定性。

1. 问题背景

考虑常微分方程初值问题 (Initial Value Problem, IVP):

\[ \begin{cases} \frac{dy}{dx} = f(x, y) \\ y(x_0) = y_0 \end{cases} \]

其中 \(f: [a, b] \times \mathbb{R} \to \mathbb{R}\) 是已知函数,\(x_0 \in [a, b]\) 是初值点,\(y_0\) 是初值。

目标:在区间 \([x_0, X]\) 上求解该方程,得到数值解 \(\{y_n\}\) 近似精确解 \(y(x_n)\)。

2. Euler法的构造

2.1 基本思想

在区间 \([x_0, X]\) 上取步长 \(h\),将区间等分为:

\[ x_n = x_0 + nh, \quad n = 0, 1, 2, \ldots, N, \quad X = x_0 + Nh \]

几何解释:精确解 \(y(x)\) 是一条光滑曲线,在每一点 \((x_n, y_n)\) 处,曲线的切线斜率为 \(f(x_n, y_n)\)。Euler法用折线段代替曲线,即用切线近似曲线。

2.2 显式Euler公式的推导

从 \((x_n, y_n)\) 出发,沿切线方向前进一步:

\[ \boxed{y_{n+1} = y_n + h f(x_n, y_n)} \]

这就是显式Euler公式 (Explicit/Forward Euler Method)。

算法步骤:

  1. 初始化:\(y_0 = y(x_0)\)
  2. 对 \(n = 0, 1, 2, \ldots, N-1\):
    • 计算 \(y_{n+1} = y_n + h f(x_n, y_n)\)
    • \(x_{n+1} = x_n + h\)

2.3 隐式Euler公式的推导

如果用点 \((x_{n+1}, y_{n+1})\) 处的切线斜率来构造格式:

\[ \boxed{y_{n+1} = y_n + h f(x_{n+1}, y_{n+1})} \]

这就是隐式Euler公式 (Implicit/Backward Euler Method)。

特点:\(y_{n+1}\) 出现在等式两边,需要用迭代法(如牛顿法)求解。

3. 截断误差分析

3.1 局部截断误差

定义 3.1 (局部截断误差)
假设在 \(x_n\) 处精确解是准确的,即 \(y_n = y(x_n)\),则从 \(x_n\) 到 \(x_{n+1}\) 的一步计算的误差称为局部截断误差 (Local Truncation Error):

\[ R_n = y(x_{n+1}) - y_{n+1} \]

定理 3.1
对于显式Euler法,局部截断误差为:

\[ R_n = \frac{h^2}{2} y''(\xi_n), \quad \xi_n \in (x_n, x_{n+1}) \]
即 \(R_n = O(h^2)\)。

证明:

假设 \(y_n = y(x_n)\) (前一步精确),对精确解 \(y(x)\) 在 \(x_n\) 处进行Taylor展开:

\[ y(x_{n+1}) = y(x_n + h) = y(x_n) + h y'(x_n) + \frac{h^2}{2} y''(\xi_n) \]
其中 \(\xi_n \in (x_n, x_{n+1})\) (Taylor定理余项)。

由微分方程 \(y'(x_n) = f(x_n, y(x_n))\),且 \(y_n = y(x_n)\),有:

\[ y(x_{n+1}) = y_n + h f(x_n, y_n) + \frac{h^2}{2} y''(\xi_n) \]

而Euler法给出:

\[ y_{n+1} = y_n + h f(x_n, y_n) \]

两式相减:

\[ R_n = y(x_{n+1}) - y_{n+1} = \frac{h^2}{2} y''(\xi_n) = O(h^2) \quad \square \]

3.2 整体截断误差

定义 3.2 (整体截断误差)
从初值 \(y_0\) 开始,经过 \(n\) 步迭代积累的总误差:

\[ \epsilon_n = y(x_n) - y_n \]

定理 3.2
在区间 \([x_0, X]\) 上,若 \(|y''(x)| \le M\),则Euler法的整体截断误差满足:

\[ |\epsilon_n| \le \frac{M}{2L} \left( e^{L(x_n - x_0)} - 1 \right) h = O(h) \]
其中 \(L\) 是Lipschitz常数。

证明:

设 \(\epsilon_n = y(x_n) - y_n\),则:

\[ \epsilon_{n+1} = y(x_{n+1}) - y_{n+1} \]

分解为:

\[ y(x_{n+1}) = y(x_n) + h f(x_n, y(x_n)) + R_n \]
\[ y_{n+1} = y_n + h f(x_n, y_n) \]

两式相减:

\[ \epsilon_{n+1} = \epsilon_n + h[f(x_n, y(x_n)) - f(x_n, y_n)] + R_n \]

由Lipschitz条件 \(|f(x, u) - f(x, v)| \le L|u - v|\):

\[ |\epsilon_{n+1}| \le |\epsilon_n| + hL|\epsilon_n| + |R_n| \]
\[ |\epsilon_{n+1}| \le (1 + hL)|\epsilon_n| + |R_n| \]

由定理3.1,\(|R_n| \le \frac{Mh^2}{2}\)。递推得:

\[ |\epsilon_n| \le (1 + hL)^n |\epsilon_0| + \frac{Mh^2}{2} \sum_{k=0}^{n-1} (1 + hL)^k \]

初值精确 \(\epsilon_0 = 0\),几何级数求和:

\[ |\epsilon_n| \le \frac{Mh^2}{2} \cdot \frac{(1 + hL)^n - 1}{hL} = \frac{Mh}{2L} [(1 + hL)^n - 1] \]

由 \((1 + hL)^n \le e^{nhL} = e^{L(x_n - x_0)}\) (当 \(h\) 充分小时):

\[ \boxed{|\epsilon_n| \le \frac{Mh}{2L} \left( e^{L(x_n - x_0)} - 1 \right)} \quad \square \]

结论:Euler法的整体截断误差是 \(O(h)\),比局部截断误差 \(O(h^2)\) 低一阶,因此称为一阶方法。

4. 收敛性证明

4.1 收敛性的定义

定义 4.1 (收敛性)
如果当步长 \(h \to 0\) 时,对固定的 \(x_n = x_0 + nh\)(保持 \(x_n\) 固定),有:

\[ \lim_{h \to 0} y_n = y(x_n) \]
则称该数值方法是收敛的。

4.2 主要定理

定理 4.1 (Euler法的收敛性)
若函数 \(f(x, y)\) 在区域 \(D = \{(x, y) : |x - x_0| \le a, |y - y_0| \le b\}\) 内:

  1. 连续
  2. 关于 \(y\) 满足 Lipschitz条件:\(|f(x, y_1) - f(x, y_2)| \le L|y_1 - y_2|\) 固定 \(x_n\),令 \(h \to 0\)(意味着 \(n \to \infty\) 但 \(nh = x_n - x_0\) 保持不变):

\[ \lim_{h \to 0} |\epsilon_n| \le \lim_{h \to 0} \frac{Mh}{2L} \left( e^{L(x_n - x_0)} - 1 \right) = 0 \]

因此:

\[ \lim_{h \to 0} |y(x_n) - y_n| = 0 \quad \square \]

4.3 收敛阶

由误差估计式可知:

\[ \max_{0 \le n \le N} |y(x_n) - y_n| = O(h) \]

因此Euler法是一阶收敛的。

5. 绝对稳定性分析

5.1 显式欧拉法 (Explicit Euler)

测试方程：

\[ y' = \lambda y \]

递推公式： 代入显式欧拉公式 \(y_{n+1} = y_n + h f(x_n, y_n)\)：

\[ y_{n+1} = y_n + h \lambda y_n = (1 + h\lambda) y_n \]

增长因子：

\[ G(z) = 1 + z, \quad z = h\lambda \]

稳定性条件： 要求 \(|G(z)| \leq 1\)，即：

\[ |1 + h\lambda| \leq 1 \]

  • 实数域分析（\(\lambda\) 为实数）：

    \[ -1 \leq 1 + h\lambda \leq 1 \implies -2 \leq h\lambda \leq 0 \]
    即步长必须满足：
    \[ h \leq \frac{2}{|\lambda|} \]

  • 复平面分析（\(\lambda\) 为复数）： 稳定域是复平面上以 \((-1, 0)\) 为圆心，半径为 1 的圆盘。

结论：显式欧拉法是条件稳定的。对于刚性问题（\(|\lambda|\) 很大），必须取极小的步长。

5.2 隐式欧拉法 (Implicit Euler)

递推公式： 代入隐式欧拉公式 \(y_{n+1} = y_n + h f(x_{n+1}, y_{n+1})\)：

\[ y_{n+1} = y_n + h \lambda y_{n+1} \]
整理得：
\[ y_{n+1} (1 - h\lambda) = y_n \implies y_{n+1} = \frac{1}{1 - h\lambda} y_n \]

增长因子：

\[ G(z) = \frac{1}{1 - z}, \quad z = h\lambda \]

稳定性条件： 要求 \(|G(z)| \leq 1\)，即：

\[ \left| \frac{1}{1 - h\lambda} \right| \leq 1 \implies |1 - h\lambda| \geq 1 \]

  • 复平面分析： 稳定域是复平面上以 \((1, 0)\) 为圆心，半径为 1 的圆外区域。 显然，整个左半复平面（\(\text{Re}(z) < 0\)）都在稳定域内。

结论：隐式欧拉法是 A-稳定 (Unconditionally Stable) 的。无论步长 \(h\) 取多大（只要 \(\text{Re}(\lambda) < 0\)），数值解都能保持稳定衰减。
